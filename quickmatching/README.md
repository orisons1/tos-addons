## [QUICKMATCHING]


### 기능 소개

- 명령어를 통해 자동매칭 챌린지 매칭을 자동화
- 자동매칭 챌린지 입장 후 캐릭터 별로 설정된 챌린지 버프를 자동 선택 ( 설정하지 않았다면 선택하지않음 )


### 명령어 및 사용 방법

#### 자동매칭
- ``/qm 2`` 자동매칭 챌린지 2단계
- ``/qm 3`` 자동매칭 챌린지 3단계
- ``/qm 4`` 자동매칭 분열 챌린지

#### 자동버프
- ``/qm buff [개인버프숫자] [파티버프숫자]`` 
- 개인버프숫자 : 왼쪽부터 1~5
- 파티버프숫자 : 왼쪽부터 1~4
- ex) /qm buff 3 1 : 스플릿 + 파티증뎀
