# Quick Path Finder

## Function

- 바운티 헌트 시스템에서 목적지까지 경로 제공
- 다음 이동할 맵 이름 및 이동해야할 맵 갯수 표시
- 현재 맵에서의 캐릭터 위치 및 이동해야 할 포탈 위치 표시 
  (m 키눌러서 켜지는 미니맵 아니고 바운티 헌트 창 밑에 만들어진 맵)
<br/>
<br/>
