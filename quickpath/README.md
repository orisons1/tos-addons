## [QUICKPATH]


### 기능 소개

- 바운티 헌트 시스템에서 목적지까지 최단 루트 제공
- 전체 진행 상황을 이동한 맵 단위로 표시

### 명령어 및 사용 방법

- ``/qp on``  Enable QuickPath addon (Default)
- ``/qp off`` Disable QuickPath ddon

- 따로 UI를 제공하지 않고 바운티 헌트 시스템의 Frame의 크기를 변경시켜 사용
