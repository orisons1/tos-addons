## [QUICKPATH]


### 기능 소개

- 바운티 헌트 시스템에서 목적지까지 경로 제공
- 다음 이동할 맵 이름 및 이동해야할 맵 갯수 표시
- 현재 맵에서의 캐릭터 위치 및 이동해야 할 포탈 위치 표시

### 명령어 및 사용 방법

- 명령어를 지원하지 않습니다.
- 다른 addon을 사용하고 계신다면 bin 폴더에서 ipf 파일다운 받아  ?:\Nexon\TreeOfSavior\data\ 경로에 ipf 파일을 설치하시면 동작합니다.
