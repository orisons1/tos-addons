## [QUICKTRADE]

### 기능 소개

  명령어를 통해 타 유저에게 거래를 신청할 수 있습니다.



### 명령어 및 사용 방법

- ``/거래 [팀명]`` or ``/qt [팀명]``

#### 동작
- object 출현 거리 내에서 거래 신청 

#### 동작 시점
- 채팅창에 명령어를 입력했을 때
